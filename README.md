# 💪 Lift-Phit V2

A modern fitness tracking web application designed to help users build strength, stay consistent, and eliminate guesswork in their workouts.

This is a complete rebuild of the original Lift-Phit app using updated technologies, improved architecture, and modern development best practices.

---

## 🚀 Overview

Lift-Phit V2 provides structured workout tracking with a focus on simplicity and efficiency.  
It is built specifically for beginner lifters who want a clear path to progress without overcomplicating their training.

---

## 🧱 Tech Stack

### Frontend
- React (Vite)
- React Router
- JavaScript (ES6+)
- CSS (custom / utility-based)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Other Tools
- Git & GitHub
- REST API architecture

---

## ✨ Features

- 🏋️ Track workouts and exercises
- 📊 Monitor strength progression over time
- 🗂 Organized workout sessions
- 🔐 User authentication (planned)
- 📱 Responsive, mobile-first design
- ⚡ Fast development with Vite

## 🎯 Project Goals

- Simplify workout tracking for beginners
- Automate progressive overload
- Provide a clean and distraction-free user experience
- Build a scalable, production-ready architecture
