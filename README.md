# 💪 Lift-Phit V2

A modern fitness tracking web application built with a clean, scalable stack.  
This is a complete rebuild of the original Lift-Phit app using updated technologies and best practices.

---

## 🚀 Overview

Lift-Phit V2 is designed to help users track workouts, stay consistent, and improve their fitness journey.  
This version focuses on performance, scalability, and a modern developer experience.

---

## 🧱 Tech Stack

### Frontend
- React (Vite)
- React Router
- JavaScript (ES6+)
- CSS (custom / utility-based)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Other Tools
- Git & GitHub
- REST API architecture

---

## ✨ Features

- 🏋️ Track workouts and exercises
- 📊 View progress over time
- 🗂 Organized workout sessions
- 🔐 User authentication (planned / optional)
- 📱 Responsive design (mobile-first)
- ⚡ Fast development with Vite
