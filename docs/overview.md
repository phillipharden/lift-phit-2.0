# 💪 Lift-Phit V2 – Overview

## 📌 What is Lift-Phit?

Lift-Phit is a fitness tracking application designed specifically for beginner powerlifters.  
It provides structured workout routines, tracks progress, and automates progressive overload to help users build strength efficiently.

---

## 🎯 Purpose

Many beginner lifters struggle with:
- knowing where to start
- tracking workouts consistently
- understanding progressive overload

Lift-Phit solves this by:
- providing pre-built workout routines
- automatically increasing weights over time
- organizing workout data in a clean, structured way

---

## 👤 Target Audience

- Beginner lifters (high school → college age)
- Busy individuals with limited gym time
- Users who want a simple, guided workout experience

---

## 🧩 Core Problem

Traditional methods (notes apps, paper logs):
- are messy
- lack structure
- require manual planning

---

## ✅ Solution

Lift-Phit provides:
- guided workouts
- automatic progression
- structured tracking
- minimal decision-making

---

## 🏆 Key Value

> “Show up, follow the plan, get stronger.”

Lift-Phit removes the guesswork so users can focus on training.
